# OverridePageHeader

一个可覆写的`PageHeader`组件，主要用于分离母版布局和参数设置

> 例如:

我在`Layout`中指定`PageHeader`的位置，然后可以在具体页面代码的任意位置使用`OverridePageHeader`来覆写具体参数。
